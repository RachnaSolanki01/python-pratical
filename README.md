# E-Library Data Insights Dashboard

## About the Project

This project is developed using Python to analyze library transaction data. It helps in finding useful information such as the most borrowed books, active users, borrowing trends, and genre distribution.

The project uses Pandas, NumPy, Matplotlib, and Seaborn for data analysis and visualization.

## Features

* Load data from CSV file
* Check for missing values
* Find most borrowed book
* Find most active user
* Calculate average borrowing duration
* Filter data by genre
* Generate simple reports
* Create charts and visualizations

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Dataset

The dataset used in this project contains the following fields:

* Transaction ID
* Date
* User ID
* Book Title
* Genre
* Borrowing Duration (Days)

## Visualizations

The project includes:

1. Bar Chart for most borrowed books
2. Line Graph for monthly borrowing trends
3. Pie Chart for genre distribution
4. Heatmap for borrowing activity

## How to Run

1. Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn
```

2. Open Jupyter Notebook.

3. Run all notebook cells in sequence.
